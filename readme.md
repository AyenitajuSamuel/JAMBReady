## JAMBReady
A smart JAMB exam preparation platform built with Django — combining exam simulation, performance tracking, and personalized recommendations.

JAMBReady is a web-based application designed to help Nigerian students prepare effectively for the Joint Admissions and Matriculation Board (JAMB) exam.

It goes beyond simple practice by providing:

-Realistic CBT exam simulations
-Structured question practice
-Performance tracking
-Intelligent, data-driven recommendations

The goal is simple: help students study smarter and improve their chances of success.

## Tech Stack
Backend: Django
Frontend: Django Templates (HTML, CSS)
Database: SQLite
Language: Python
